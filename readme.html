Password Calculator

Calculates a password from a master password, your name and the site name.

Start using it

What is it?

Password Calculator takes a master password, your name and a site name to calculate a password. The password - and of course the master password - will never be stored, it will always be calculated.

This means, you do not have to save a key store file. Every password can be calculated any time, anywhere. This is possible because you know the necessary information.

Your name + Master password + Site name = Your password for the site

The details

In the key section you specify your name, which can be saved, and your master password. As soon as you change your master password, all site passwords will be calculated.

All your saved sites are listed in the main area. Click on a header to expand a site.

A new site can be added in the "New site" panel. There you specify the site name (I recommend using one naming schema, google.com or www.google.com or https://www.google.com), a template and a counter. The counter should be incremented/set when you need a new password for the site.

A template defines how many characters and which characters a password will have:

Default & long32 latin characters, numbers and some special characters
Default & medium10 latin characters, numbers and some special characters
Default & short6 latin characters, numbers and some special characters
Simple & long32 latin characters and numbers
Simple & medium10 latin characters and numbers
Simple & short6 latin characters and numbers
Pin, medium10 Numbers
Pin, short4 Numbers

Storage

NO PASSWORD WILL BE STORED! Never, ever! For your convenience only site configuration will be stored, optional your name.

Currently, your browses local storage will be used. You can export and import your configuration in export/import panel and transfer it to another device. As you can always reconstruct your passwords, this is only for your convenience.

A configuration is a JSON document and looks like this: 

[
	{
		"name":"google.com",
		"template":"default-long",
		"counter":"1"
	},
	{
		"name":"derstandard.at",
		"template":"pin-short",
		"counter":"1"
	}
]

There is no secret, you can even safely publish this information like I do in this example. With this information they cannot calculate your site's password or even worse your master password.
